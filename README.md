# 🏃‍♂️ Nike Sales Performance Dashboard

## 📌 Project Overview

This project analyzes Nike sales data and builds an interactive Power BI dashboard.

---

## 🛠️ Tools Used

* Python (Pandas)
* Google Colab
* Power BI

---

## 🧹 Data Cleaning

* Handled missing values
* Fixed inconsistent regions
* Removed negative values
* Standardized date formats

---

## 📊 Dashboard Features

* KPI Cards
* Product Analysis
* Region Analysis
* Monthly Trend
* Channel Comparison

---

## 💡 Key Insights

* Lifestyle generates highest revenue
* Training is most profitable
* Men contribute highest revenue
* Mumbai is top region
* Online channel performs strong
